# What easyPublisher Can Do

- **Markdown support**  
  Headings, lists, links, highlights, quotes, code

- **Internal linking**  
  `[[link]]` syntax for seamless navigation

- **Plugin system**  
  Toolbar buttons, text tools, highlights, export, more

- **Local storage**  
  Notes and custom data — saved automatically

- **Themes and JS plugins**  
  Drop files into the `/theme` folder. No setup.

- **Blazing fast**  
  No build step. No framework. No overhead.

---

> easyPublisher isn’t a CMS. It’s a writing tool.
